# Contributing Guidelines

Thank you for your interest in contributing to this repository!
To ensure content quality and consistency, please follow these rules.

1. **Correct Location**: 
   All content must follow the structure: `/{category}/{type}/{format}/{filename}`.
   
   - **Category**: Game or project name in lowercase (e.g., `cs2`, `minecraft`, `web_assets`).
   - **Type**:
     - `images/`: For static images
     - `videos/`: For video files
     - `gifs/`: For GIF animations
   - **Format**: Specific folder for the file extension (e.g., `png`, `avif`, `mp4`, `gif`).

2. **Supported Formats**:
   - **Images**: `png`, `jpg`, `avif`, `webp`
   - **Video**: `mp4`, `webm`
   - **GIF**: `gif`

3. **Filenames**:
   - Use exclusively **snake_case** (lowercase and underscores).
   - No spaces or special characters.
   - Example: `my_new_file.png`, `wallpaper_hd.mp4`.

4. **Quality and Content**:
   - Ensure the content is relevant and of acceptable quality.
   - Do not upload files with intrusive watermarks or offensive content.
   - If it's Workshop map content, consider adding the link in the PR description.
